# Vanta City title and loading art

These two original, project-owned backgrounds were generated for
PRESENTATION-002 through the official OpenAI Image API using the repository's
explicitly approved `gpt-image-2` CLI path. Runtime uses only the local JPEG
derivatives; there is no runtime network request.

- `northbar-title-hero.jpg` is decorative title art. Its arriving traveler is a
  generic figure, not a face-accurate portrait of Rook. All title, setting,
  story, control, and accessibility text remains semantic HTML.
- `junction-loading-road.jpg` is decorative loading art. Progress, readiness,
  failure, fallback, and elapsed-time truth remains DOM content driven by the
  existing runtime authorities.
- `sources/` contains the two accepted API outputs. The two rejected candidates
  were inspected but are not committed.

The accepted work is painterly screenprint/rotogravure illustration with broad
coastal-industrial geometry. It does not claim exact low-poly character
identity. Full prompts, request settings, review decisions, transformations,
dimensions, and hashes are recorded in `provenance.json`.
